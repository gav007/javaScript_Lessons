JavaScript Foundations Reps

Open a lab, read the README, complete the TODOs, and keep your fingers moving.

Make buttons change text, count clicks, toggle a panel, and reset the page.

variables, numbers, functions, events, disable buttons

input values, trim, basic validation, template strings

select dropdowns, CSS variables, style.setProperty, change events

numbers, parseFloat, NaN checks, functions, parameters, returns

05-quiz-engine · Quiz Engine

Build a small quiz that moves through questions and tracks score.

arrays, objects, indexes, rendering, conditionals

06-array-list-render · Array List Render

Render a list of products/cards from an array of objects.

arrays, objects, forEach, createElement, append

07-card-filter-search · Card Filter Search

Filter cards as the user types into a search box.

input events, filter, includes, lowercase, rendering

08-todo-list-lite · Todo List Lite

Add, complete, and delete tasks without local storage yet.

CRUD, arrays, objects, forms, event.preventDefault, rendering

localStorage, JSON.stringify, JSON.parse, loading saved data

10-modal-tabs-accordion · Modal Tabs Accordion

UI state, classList, event delegation, data attributes

11-form-validation · Form Validation

forms, validation, error messages, email patterns, submit events

12-text-to-speech · Text To Speech

Convert typed text into speech, with voice, pitch, and rate controls.

Web Speech API, select options, sliders, browser APIs, objects

13-fetch-json-users · Fetch JSON Users

Fetch sample user data and render it as cards.

fetch, async/await, JSON, try/catch, loading states

14-weather-app-shell · Weather App Shell

Build the shell for a weather app and learn where the API code belongs.

API keys, fetch, async/await, JSON parsing, error handling

15-tic-tac-toe · Tic Tac Toe

Build browser tic-tac-toe with a 3x3 board and win checking.

arrays, game state, win conditions, loops, functions

16-soundboard-vanilla · Soundboard Vanilla

Build a simple soundboard using plain JavaScript before touching p5.js.

Audio object, keyboard events, data attributes, media files
